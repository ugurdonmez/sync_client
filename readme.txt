1. İlk çalıştırmada install.sh in çalıştırması gerekiyor. Gerekli python ve samba paketleri kurmak için. 

2. Sonraki çalıştırmalarda start.sh çalıştırılır. 

3. "Folder" içinde yapılan değişiklikler samba server indaki yere anlık olarak aktarılır. 

4. Samba server ip si 192.168.1.200 olmalıdır.
